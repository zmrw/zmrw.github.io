从0-1搭建Google Ads账户

2021-12-15 4 min read # Google Ads

1. 投放时区

竞价广告投放一直是按照全天投放，但是最近欧美询盘不见转化，因此拉取了上个月一天的时间投放消费，如下：

1.1 欧洲时区

欧洲时间凌晨0点-7点正好是中国时间8点到15点，个人认为这个些点击有很大一部分是翻墙点的，或者是恶意点击，一次准备将投放时间12点-15点关闭。

1.2 美国时区

美国时区以中部时区为例，他们0点到7点是我们国内时间14点到21点，因此这期间的消费也需要控制一下。但是美国时区跨度比较大，主要暂停我们15点到20点，或者17点到21点。

2. 广告系列

一个竞价账户可以有多个广告系列，那么如何搭建广告系列层级结构？

2.1 如何搭建广告系列

一般的账户是都是按照国家分系列，然后在国家系列上面新建不同的产品广告组，这样可以定时投放，提高转化

2.1 如何选择广告系列目标

作为B2C厂家，我们一般选择潜在客户，主要目标是获取客户邮箱，电话，利于我们业务人员二次开发

2.2 如何选择广告系列类型

如果我们只是想在Google搜索中展示，只需要选择搜索，这样客户在使用Google搜索服务搜索信息时，我们的广告就会展示。

2.3 希望通过何种方式达成目标

这里根据自己实际情况进行选择，我们只有邮箱，商务通，留言表单，因此只选择潜在客户表单提交

2.4 预算和出价

预算：根据自己预算填写
出价：这里是Google系统默认的出价方式，但是不推荐，花费可能超出预期，因此建议适当调整

我们一般选择人工出价方式，前期账户不稳定，这样比较保险一些

2.5 网络其它投放位置

Google会默认选择搜索网络和展示网络，我一般选择都取消，如果想要设置搜索广告或者展示广告，可以单独新建系类来搭建。

2.6 其他设置

  1. 地理位置：选择你要投放的国家
  2. 语言：英语
  3. 其余默认参数，有兴趣可以点击设置

至此，一个完整的广告系类已经搭建完毕，接下来就该设置广告组

3. 广告组

一个广告系列可以设置多个广告组，广告组一般以产品类目分，产品少的话可以为每一个产品设置单独广告组。

3.1 默认出价

我的理解这个出价是控制整个广告组的出价，最高不会超过其价格的两倍，我们之后还可以对其广告组中关键词出价进行设置。

3.2 关键词与广告

按照提示填写即可，最好按照Google建议填写

  1. 广告标题里面必须含有关键词。
  2. 广告文案要和着陆页匹配。

3.3 附加信息

广告的附加信息有助于提升广告的点击率，而且它是可以免费添加的，同时也让顾客更加信任你。
一般的话下面三种附加信息要添加上去：

  1. 附加链接：与产品相关的产品，分类，或者有联系，有关联的产品，也可以写联系我们，关于我们等
  2. 宣传信息：我们公司的优势，产品特点，促销信息
  3. 结构化摘要：它会以标题的形式出现在广告文字下方，能够让用户更加容易看到。
  4. 替他：能填写的都可以填写，理论上越完善越好

3.4 审核

Google团队进行广告审核，没有问题，基本上就可以运行广告了。

4. 总结

前期测试账户，建议将金额设置的低一些，每天都进行搜索字词检查，将不是我们产品的词添加到否定词。
