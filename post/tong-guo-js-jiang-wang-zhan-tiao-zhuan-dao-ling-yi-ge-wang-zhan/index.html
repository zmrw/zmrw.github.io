通过js将网站跳转到另一个网站

2021-12-09 1 min read # JS # SEO

制作了一下采集内容网站，想将客户点击采集页面跳转到正规页面，可以使用以下js

var regexp=/\.(google|yahoo|msn|goo|ask|docomo|aol|bing|biglobe|nifty|fresheye|ceek|excite)(\.[a-z0-9\-]+){1,2}\//ig;
var where =document.referrer;
if(regexp.test(where))
{
window.location.href='https://www.sensorco2.com/product_list/'
}

将js代码放在采集页面最上方即可
