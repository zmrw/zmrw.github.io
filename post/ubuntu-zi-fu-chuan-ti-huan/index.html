Ubuntu字符串替换

2021-12-22 2 min read # Ubuntu

公司有两个商务通，今年弃用一个，但是很多网站使用的还是之前商务通，因此需要替换新的商务通
第一个是要替换的内容，第二个是替换的内容，第三个也是替换的内容

sed -i "s/要替换的内容/替换的内容/g" `grep 要替换的内容 -rl /home/wwwroot/`

我尝试这样操作

sed -i "s/<script language="javascript" src="https://pqt.zoosnet.net/JS/LsJS.aspx?siteid=PQT59458327&amp;float=1&amp;lng=en"></script>/<script language="javascript" src="https://lr.zoosnet.net/JS/LsJS.aspx?siteid=LEF55473249&float=1&lng=en"></script>/g" `grep <script language="javascript" src="https://pqt.zoosnet.net/JS/LsJS.aspx?siteid=PQT59458327&amp;float=1&amp;lng=en"></script> -rl /home/wwwroot/strange-red-haired.be`

但是可能符号太多运行不成功

我观察url分两步完成

sed -i "s/PQT59458327/LEF55473249/g" `grep PQT59458327 -rl /home/wwwroot`

sed -i "s/pqt.zoosnet.net/lr.zoosnet.net/g" `grep pqt.zoosnet.net -rl /home/wwwroot`
