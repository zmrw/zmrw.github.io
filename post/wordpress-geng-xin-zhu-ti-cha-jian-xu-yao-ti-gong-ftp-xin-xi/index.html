wordpress更新主题插件需要提供ftp信息

2021-12-14 2 min read # Wordpress

第一次在ubuntu安装wordpress,更新主题时，提示

To perform the requested action, WordPress needs to access your web server. Please enter your FTP credentials to proceed. If you do not remember your credentials, you should contact your web host.

原因是php-fpm的用户并没有wordpress解压目录的写权限

1.查看php-fpm的用户是谁

ps -ef | grep "php-fpm"

结果如下：

root@ubuntu:/home/lnmp1.8# ps -ef | grep "php-fpm"
root      3848     1  0 09:49 ?        00:00:00 php-fpm: master process (/usr/local/php/etc/php-fpm.conf)
www       3850  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3851  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3852  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3853  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3854  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3855  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3856  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3857  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3858  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3859  3848  0 09:49 ?        00:00:00 php-fpm: pool www
www       3891  3848  0 09:50 ?        00:00:00 php-fpm: pool www
root      3949 18961  0 09:56 pts/2    00:00:00 grep --color=auto php-fpm

这里用户为www

2.修改wordpress解压所在的目录权限

sudo chown -R www /home/wordpress/wordpress.keinder.nl

sudo chown -R www 加上部署路径
重新更新主题，插件即可
